# KC3Hack 2021 Template

[![KC3Hack](https://kc3.me/hack/wp-content/uploads/2021/01/kc3hack2021ogp@2x.png)](https://kc3.me/hack)

## プロダクト名
BookTree!

## プロダクト説明
読書生活を豊かにするwebアプリです．

## 使用技術
```
- Vis.js
- Rust
- Actix Web
- CSS
- Docker
```
## 実行方法
1. リポジトリをclone <br>
2. ディレクトリを移動
```
cd server
```
<br>

3. コンテナを起動
```
docker-compose up --build
```
