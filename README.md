## 使用技術
- Vis.js
- Rust
- Actix Web
- CSS
- Docker
- Google Books APIs


## 実行方法
1. リポジトリをclone <br>
2. ディレクトリを移動
```
cd server
```
<br>

3. コンテナを起動
```
docker-compose up --build
```
<br>

4. サイトにアクセス
```
http://localhost:8080
```
